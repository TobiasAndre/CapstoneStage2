## Android Developer Nanodegree

# Capstone Stage 2
This application is part of the final project of the Android Nanodegree course.

## Features
With the app, you can:
* Add Customers Info.
* Add Treatments Info
* Schedule Beauty treatments
* Show reports & Graphics
